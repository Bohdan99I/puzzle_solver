# Puzzle Solver

## Опис

Цей проект призначений для розв'язання завдання зі складання найбільшого однорядного цифрового пазлу.

## Використання

1. Встановіть залежності:

```bash
npm install
```

2. Запустіть проект:

```bash
npm start
```
